# 🛒 Amazon UI Clone

A simple and responsive **Amazon Homepage Clone** developed using **HTML5** and **CSS3**. This project was built to improve my front-end development skills by recreating the design and layout of one of the world's most popular e-commerce websites.

---

## 🚀 Features

- Amazon-inspired homepage layout
- Responsive navigation bar
- Search bar with category selector
- Hero banner section
- Product category cards
- Shopping cart icon
- Footer with multiple sections
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons

```bash

```

### Run the Project

Open the **index.html** file in your preferred web browser.
